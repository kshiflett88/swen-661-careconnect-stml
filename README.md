## Project Description
This project redesigns the CareConnect user interface to support care recipients with Short-Term Memory Loss (STML). The application emphasizes guided, structured workflows, persistent orientation cues, reduced cognitive load, and accessibility-first design while maintaining functional parity with the baseline system.
